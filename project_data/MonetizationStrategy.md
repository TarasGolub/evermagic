# EverMagic — Launch Plan

## Goal
First 10 paid orders + 5 reviews → Etsy algorithm starts working

**Constraints:** ~5 hrs/week · $50–200/mo marketing budget

---

## Week 1 — Etsy Launch (3 hrs)

- [ ] Create Etsy shop (if not done)
- [ ] Create coupon `FAMILY75` (75% off, single-use per account)
- [ ] Mockup in Canva — book "in hands", not PDF screenshot
- [ ] Listing 1: Basic (PDF storybook only)
- [ ] Listing 2: Full Bundle (storybook + certificate + coloring)
- [ ] Listing 3: Party Pack
- [ ] SEO titles: "Personalized [Theme] Story Book for Kids – Custom Children's Book Gift"
- [ ] Add 13 tags per listing (mix: "personalized kids book", "custom story gift", "children birthday gift")
- [ ] Set processing time: 1–2 business days
- [ ] Publish all 3 listings

---

## Week 2–3 — First 10 Orders (2 hrs)

- [ ] Personal outreach to 10 friends/family — send coupon FAMILY75
- [ ] Ask them to order on different days (2–3 per day, not all at once)
- [ ] After delivery, send personal message asking for a *text* review (not just stars)
- [ ] Target: 5 text reviews minimum
- [ ] Post in Edmonton parent Facebook groups — organic, no spam

**Review ask template (copy-paste):**
> "Hey! Would you mind leaving a review on Etsy? Even a sentence or two about what you liked helps a lot — it's how new shops get discovered. Here's the link: [link]"

---

## Week 4 — Analyze + First Paid Traffic (1–2 hrs)

- [ ] Check Etsy Stats: views, clicks, conversion rate
- [ ] If conversion rate > 2% → turn on Etsy Ads, $2–3/day budget
- [ ] If conversion rate < 2% → fix listing (photos or description) before spending on ads
- [ ] Etsy Ads budget: $50–100/mo (not Facebook — Etsy buyers already have intent to buy)

---

## Month 2 — After First Revenue

- [ ] Sign up for Gelato (free) + order sample copy of current PDF
- [ ] Add Claid API upscaling (300 DPI) to Image Generation n8n workflow
- [ ] Refactor PDF templates to 8.5×8.5" + 0.125" bleeds for Gelato specs
- [ ] New Etsy listing: "Printed Storybook + Digital Bundle" — $35–45 CAD

---

## Month 3 — Expand

- [ ] Analyze which theme sells best → double down on it
- [ ] Add Pet + Kids theme ("Child and dog on adventure together")
- [ ] Build n8n Workflow 4 — Gelato POD fulfillment automation
- [ ] Evaluate: own landing page + Stripe if Etsy traffic is consistent

---

## Parking Lot (not now)

- Amazon KDP
- Mobile app
- Mini cartoon/video
- Book series without personalization
- Pinterest / TikTok (only if time allows after Month 2)

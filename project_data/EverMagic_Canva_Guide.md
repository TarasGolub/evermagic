# EverMagic — Etsy Listing Image Guide for Canva

## Setup

**Canvas size:** 1080 × 1080 px (Canva square format — Etsy recommended)
**Font:** Poppins (available free in Canva)
**Workflow:** Upload the matching clean base image as the background layer, then add text + element layers on top.

## Canva Workflow — Step by Step

1. Open Canva → **Create a design** → Custom size → `1080 × 1080 px`
2. Upload the base image for the theme you're working on
3. Set it as **Background** (right-click → Set as background) — or place it as a full-bleed element locked at the bottom
4. Add text layers one by one following the specs above
5. Use **Poppins** from Canva's font library (search "Poppins")
6. Use the hex color codes in the table at the top — paste them into the color picker
7. Export: **Download → JPEG → Quality 100%**

**Naming convention for uploads to Etsy:**
```
01_hero_mockup_{theme}.jpg
02_spread_pages_{theme}.jpg
...
```
